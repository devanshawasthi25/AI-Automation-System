# AI Automation System

An AI-powered automation platform built with **FastAPI (Python)** and **React (Vite)** that allows users to run automated research, web scraping, and AI analysis from a simple web dashboard.

## Features
- 🔍 AI Research Agent
- 🌐 Web Scraper
- 🤖 AI-powered analysis
- ⚡ FastAPI backend
- 🎨 React frontend dashboard
- 🔗 API-based automation workflows

## Tech Stack
- Python
- FastAPI
- React (Vite)
- JavaScript
- REST APIs

## Project Structure
ai_automation_system_full
├── backend
│ ├── app
│ ├── run.py
│
├── frontend
│
└── requirements.txt


## Run Locally

### Backend
bash
- cd backend 
- pip install -r requirements.txt
- python run.py

### Frontend
bash
- cd frontend
- npm install
- npm run dev

### THEN OPEN
http://localhost:5173
